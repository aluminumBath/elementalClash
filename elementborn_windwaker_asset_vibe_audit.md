# Elementborn Wind Waker-Like Asset Vibe Audit

This audit reviews the assets present in `elementborn_combined_master_patch_v49.zip` and the additional prefabs/materials that the Unity editor generators create after you run the Elementborn menus.

Target vibe used for the audit:

```text
Legend of Zelda: Wind Waker-like fantasy
cel-shaded
bold clean outlines
simple readable silhouettes
hand-painted / storybook / nautical-fantasy feel
warm playful colors
low-poly or stylized forms
minimal realistic texture noise
minimal modern-app / mobile-game gloss
```

## Scope and limitations

```text
Patch reviewed: elementborn_combined_master_patch_v49.zip
Actual files under Assets/Elementborn: 823
C# files: 634
PNG files: 144
WAV files: 27
CSV files: 12
Unity-generated prefabs/materials: created later by menu scripts, not stored in the ZIP yet
```

I cannot open the Unity scene or generated prefabs here, so this is a source/asset audit based on the ZIP contents plus the generator scripts. The Unity compiler and scene view remain the final source of truth.

## High-level result

The current project is very strong as a **systems/playtest scaffold**, but many visual/audio assets are still **prototype, debug, or generic mobile-UI placeholders** rather than Wind Waker-like final assets.

The biggest mismatch areas are:

```text
1. generated primitive 3D landmarks and placeholder prefabs
2. modern circular/rounded-square UI icon style
3. debug/admin/playtest UI panels using flat dark boxes and Arial text
4. generated placeholder WAVs
5. setup/developer icons that should remain editor-only
```

## Priority legend

```text
P0 = should replace before visual review / public build
P1 = okay for internal testing, replace before vertical slice trailer/demo
P2 = developer/debug-only; keep hidden from player-facing builds
```

# P0: Major visual mismatches

## 1. Generated capital landmark prefabs

These are functional and useful, but they are currently generated from Unity primitives such as cylinders and cubes with simple materials. They will not feel like Wind Waker-like architecture until replaced with authored stylized meshes.

| Generated asset | Current issue | Desired replacement |
|---|---|---|
| `Assets/Elementborn/Generated/Prefabs/Capitals/FireCapital_VolcanoCitadel.prefab` | Built from stacked primitive rings, cubes, cylinders, lava strips. Reads as blockout, not fantasy architecture. | Stylized volcano city with chunky basalt shapes, painted lava channels, big silhouette, curved bridges, throne-spire, warm toon lighting. |
| `Assets/Elementborn/Generated/Prefabs/Capitals/WindCapital_SkySpire.prefab` | Cylinder/ring tower blockout. | White/ivory wind spires with cloth banners, spiral ramps, floating-feel architecture, big readable shapes. |
| `Assets/Elementborn/Generated/Prefabs/Capitals/EarthCapital_TerracePalace.prefab` | Cube terrace blockout. | Layered earthen palace with roots, gardens, carved stone, oversized soft forms. |
| `Assets/Elementborn/Generated/Prefabs/Capitals/MetalCapital_FoundryKeep.prefab` | Boxy keep/furnace blockout. | Stylized brass/steel foundry city with exaggerated smokestacks, heavy silhouettes, less realistic metal texture. |
| `Assets/Elementborn/Generated/Prefabs/Capitals/NeutralCity_ConvergenceNexus.prefab` | Primitive plaza/crystal blockout. | Warm central hub with hand-painted stone, water canals, flags, color-coded districts, less sci-fi crystal look. |

Recommended update: keep `CapitalLandmarkDescriptor` and spawn anchors, but replace primitive children with authored FBX/GLB models or polished prefabs.

# Wind Waker-like art checklist for replacements

Use this checklist for each replacement asset:

```text
clear silhouette from a distance
simple geometry, not realistic detail
bold outline or outline-like edge treatment
flat readable color blocks
warm hand-painted palette
slight asymmetry / hand-made charm
minimal noisy texture
minimal glossy modern UI gradients
fantasy/nautical motifs where appropriate
works in cel-shaded lighting
recognizable at small icon size if UI
```
