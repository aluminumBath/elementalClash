# Art Style Note for v15

## What was updated

The main patch-generated image set that most needed adjustment was the v14 equipment icon set.
Those icons were polished, but not yet strongly aligned with the project's intended Wind Waker-like visual direction.

### Updated in v15

- Refreshed the full generated equipment icon set
- Added a new combat icon set using the same cel-shaded style

## Wind Waker-like characteristics used

- strong dark outlines
- simple readable silhouettes
- bright saturated colors
- flattened cel-shaded lighting
- painterly fantasy / nautical UI tone
- stylized over realistic detail

## Not changed

This pass updates the patch-generated icon PNGs created inside the rolling master patches.
It does not overwrite user-created art or external asset art already present in the project repo.
