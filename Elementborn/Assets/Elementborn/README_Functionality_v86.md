# v86 Functionality: Filled-Out Prototype Zone Content

v86 fills out the hub so it feels more like an early game zone.

## New interactables

```text
Elemental envoys
Resource nodes
Healing shrines
Loot chests
Lore stones
```

## New progression chain

The prototype quest chain is now:

```text
Talk to Ember Guide
Choose Unifier or Dominion
Collect the glowing shard
Return the shard
Open matching elemental gate
Defeat hostile
Gather 3 elemental essence
Open Convergence Reward Chest
Read a lore stone
Zone complete
```

## Inventory/progression HUD

The HUD now tracks:

```text
Elemental Essence
Coins
Lore discovered
Chests opened
Zone Level
```

## Hub dressing

The scene now includes:

```text
Roads from the central platform
Envoys in elemental quadrants
Market stalls
Unifier and Dominion banners
Small central arch
Essence nodes in each quadrant
Healing shrines
Reward chest
Side supply chest
Lore stones
```

## Notes

This is still procedural primitive geometry, not final art, but it now gives the prototype more of Elementborn's actual game structure:
- factions/ideology
- elemental regions
- hub exploration
- interactive NPCs
- resources
- loot
- lore
- recovery points
- multi-step quest progression
