# v87 Functionality: Readable Messages + Journal

v87 improves prototype readability.

## Message timing

Before v87, transient messages could disappear too quickly.

New defaults:

```text
HUD messages: about 11 seconds
Dialogue: about 16 seconds
```

## Larger UI

The HUD message area and dialogue box are larger and easier to read.

## Recent message log

A recent-message panel now appears in the upper-right corner.

```text
M = toggle recent message log
```

It keeps the most recent messages visible so important prompts are not missed.

## Quest journal

A basic quest/progression journal is now available:

```text
J = toggle journal
```

It shows:
- current objective
- selected element
- selected stance
- quest state
- essence
- coins
- lore read
- chests opened
- zone level
- full prototype loop checklist

## Dialogue dismissal

```text
Enter = dismiss dialogue
```
