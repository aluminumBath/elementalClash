# Toon Material Notes

For fastest setup in Unity URP:

1. Use URP/Lit or Simple Lit.
2. Set Smoothness low, around 0.05–0.2.
3. Use a directional light and ambient color.
4. For stronger cel shading, use a Toon Shader Graph or a toon shader package already approved for your project.
5. Keep textures simple and saturated; the concept art style relies on clean silhouettes and hard shadow shapes.

Recommended material naming:

- `M_EB_Toon_Skin`
- `M_EB_Toon_Cloth_Red`
- `M_EB_Toon_Cloth_Blue`
- `M_EB_Toon_Stone`
- `M_EB_Toon_Water`
- `M_EB_Toon_Plant`
- `M_EB_Toon_Metal`
